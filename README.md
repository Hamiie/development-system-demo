# Clearway Release Hub

This repository deploys the public Streamlit release hub for Clearway.

Clearway is local-first. The hosted Streamlit page is the front door and update hub; the real app runs locally so it can work with the user's folders, Markdown files, backups, Google Calendar exports, and Google Tasks exports.

Deploy with Streamlit Community Cloud using:

```text
app/main.py
```

The downloadable local app source lives in `downloads/`. Build the launcher EXE locally before publishing a friend-facing package.
